# SUmusic-1.63.1

## ⚠️ 重要说明

- 本仓库代码将**不再维护**
- 当前版本内置的在线音乐 API 由作者自行搭建，仅供本版本使用
- 后续闭源版本将启用全新 API 接口，**本开源版本中的 API 地址将同步失效**
